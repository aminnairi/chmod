# chmod

Calculate `chmod` permissions using a grahpical Web interface

## Usage

See https://aminnairi.github.io/chmod

## Development

### Requirements

- Bash
- GNU/Make
- Docker
- Docker Compose

### Serve

```bash
make serve
```

### Optimize

```bash
make optimize
```
